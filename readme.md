* [Windows & MacOs app v2.20.0](https://github.com/kkamara/kelvinkamara/releases).

## Requirements

* [Node.js](https://nodejs.org/en/) (arrives with [npmjs](https://www.npmjs.com/))
* [Yarn](https://yarnpkg.com/).

## Installation

```bash
  npm i -g yarn
  # yarn
  yarn install
```

## Usage

```bash
  yarn start
```

## Building target: your machine

```bash
  yarn build
```

## Building target: cross-platform

```bash
  # --linux --win --mac --x64 --ia32
  yarn run release
```
